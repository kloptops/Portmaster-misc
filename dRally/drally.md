## Controls

| Button            | Command                    |
|-------------------|----------------------------|
| **A**             | Accelerate                 |
| **B**             | Brake                      |
| **D-Pad left**    | steer left                 | 
| **D-Pad right**   | steer right                |
| **Left Analog**   | steer right/left           |
| **R1**            | Machine Gun                |
| **R2**            | Turbo Boost                |
| **L1**            | Drop Mine                  |
| **L2**            | Horn                       |
| **Start**         | Enter                      |
| **Select**        | Back                       |
| **Select R2**     | Quick Save                 |
| **Left Analog**   | Quick Load                 |
| **Start up/down** | Text input                 |
| **R1 Select**     | tab                        |
